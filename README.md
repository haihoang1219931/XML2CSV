# Project Title

Restore Dr.Fone backup backup contacts

## Description

This code will convert Dr.Fone backup contacts to google drive's csv format

## Getting Started

### Dependencies

* Ubuntu OS
* Qt 5.11.3

### Installing

* Clone from github

### Executing program

```
./XML2VCF Contacts.list contacts-export.csv
```
